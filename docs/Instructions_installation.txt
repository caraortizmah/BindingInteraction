BindingInteraction
You need the following programs before installing BindingInteraction:

* Python 2.7 or 3
* MOPAC2016
* VMD with Dowser plugin
* Chimera
* Propka3.1
* GAMESS
* Facio (FMO software)
* Wine

For installing BindingInteractor pipeline:

Open a console and on BindingInteraction/ path, type:

1.
$ ./configure

2.
$ make init

wait... it could take several minutes

3.
$ make install

4. Create a folder on opt
$ mkdir -p /opt/BindingInteraction/

5. Copy this file on opt
$ cp BindingInteraction.pyc /opt/BindingInteraction/

6. Open your .bashrc and type at the end
$ vim .bashrc

alias BindingInteraction='/opt/BindingInteraction/indingInteraction.pyc '

7. After saving changes in bashrc
$ source .bashrc

Now yo can enjoy BindingInteraction pipeline!
