BindingInteraction

You need the following programs before installing BindingInteraction:

  • Python 2.7 or upper
  • MOPAC2016
  • VMD with Dowser plugin
  • UCSF Chimera
  • Propka 3.1
  • GAMESS
  • Facio (FMO software)
  • Wine

For installing BindingInteraction pipeline:

Open a console and type the following commands

Step Instruction Command
1 Configure program and scratch path for the installation ./configure -p 'program_path' -s 'scratch_path'
2 Install python requirements before (it could take several minutes) make require
3 Prepare folders and other programs before installing BindingInteraction make init
4 Install BindingInteraction sudo python setup.py install

 

Now you can enjoy BindingInteraction pipeline!
