REQUIREMENTS

 

The MHCBI pipeline only needs gfortran and some linux test processing tools (cut, grep, awk, gawk, sed and vim).
However the MHCBI uses several specialized softwares that rely on another kind of requirements.


Before starting any procedure regarding the MHCBI pipeline, first of all type:

    $ sudo apt update
  $ sudo apt install build-essential
  $ sudo apt install gfortran
  $ sudo apt install git


For Dowser:

You have to install csh prior to do Dowser.
Go to Dowser GitHub repository for further information.

For OpenBabel:

Go to Open Babel wiki for installing it.
Further information about releases availables at OpenBabel GitHub


For VMD:

Go to the following page for further information.
A helpful page for installing could be:

1. https://www.biostars.org/p/196147/


For Propka 3.1:

For this program you need python 3.6 or higher and pip (python package installer) for installing propka 3.1.
Go to propka 3.1 GitHub repository for further information.


For MOPAC:

Due to installing MOPAC is a straightforward process (locating properly the MOPAC executable provided at http://openmopac.net/downloads.html), you only need to configure your .bashrc or .bash_profile typing:

$ export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/opt/mopac/mopac_cuda_5.5_libs/

In this path the following files must be there:
libcublas.so.5.5 libcudart.so.5.5 libiomp5.so

Further information in the MOPAC page requirements.

 

For GAMESS:

This is a general ab initio quantum chemistry package. It is not easy to install at all.
Go to Gordon's group page for downloading the sourcecode.
For installing GAMESS there are several pages that could be useful. You can clear up any doubts reading several of those pages below:

1. https://www.webmo.net/support/gamess_linux.html

2. https://sites.google.com/site/rangsiman1993/comp-chem/program-install/install-gamess

3. http://drphilipcarlson.weebly.com/installing-gamess.html

4. http://katter-world.blogspot.com/2012/04/installing-gamess-on-ubuntu-64-bit-zz.html


For GUI Facio:

You need to install wine for executing Facio. Due to Facio is not opensource only you have to launch Facio through wine.
You can use the following page for wine:
1. https://vitux.com/how-to-install-wine-on-ubuntu/

Go to docs/Install_documentation.html for configuring and running the MHCBI pipeline.
