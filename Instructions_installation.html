BindingInteraction

You need the following programs before installing BindingInteraction:

  • Python 2.7 or 3
  • MOPAC2016
  • VMD with Dowser plugin
  • Chimera
  • Propka3.1
  • GAMESS
  • Facio (FMO software)
  • Wine

For installing BindingInteractor pipeline:

Open a console and type

Step Instruction Command
1 On BindingInteraction/ type and follow the instructions ./configure
2 type and wait (it could take several minutes) make init
3 type make install
4 create a folder on opt mkdir -p /opt/BindingInteraction/
5 copy this file on opt cp BindingInteraction.pyc /opt/BindingInteraction/
6 Open your .bashrc and type at the end alias BindingInteraction='/opt/BindingInteraction/indingInteraction.pyc '
7 After saving changes in bashrc source .bashrc

 

Now yo can enjoy BindingInteraction pipeline!
